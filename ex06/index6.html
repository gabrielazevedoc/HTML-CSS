Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag B(não semântica)

Nesta frase temos um termo em destaque usando a tag STRONG(semântica)

Itálico/Ênfase

Nesta frase temos um termo em ênfase usando a tag I(não semântica)

Nesta frase temos um termo em ênfase usando a tag EM(semântica)

Testo marcado

Podemos criar também um texto marcado usando a tag MARK

Podemos criar outras linhas com o texto marcado igual usando a tag STYLE em CSS

Texto grande e pequeno

Podemos criar um texto grande e um texto pequeno usando as tags BIG e SMALL

Texto deletado

Podemos indicar um texto excluído para falar que um texto deve ser lido mas não considerado

Texto inserido

Podemos indicar um texto inserido para dar ênfase e mostrar que ele foi colocado depois

Existe também o texto sublinhado com a tag não semântica U

Texto sobrescrito e subescrito

Podemos adicionar x20+3 e H2O com as Tags SUP e SUB
